# 🌙 오토 플래너

설정한 간격으로 트렌드 스나이퍼를 자동 실행합니다. 자는 동안 / 백그라운드에서 데이터를 누적하기 좋습니다.

## 필요한 것
- 트렌드 스나이퍼 도구 설정 완료 (API 키 등)

## 설정값
- `INTERVAL_HOURS` — 몇 시간마다 실행할지 (기본 2)
- `TOTAL_RUN_HOURS` — 총 가동 시간 (기본 8)

## 실행 방법
패널의 [▶ 실행]을 누르거나:
```bash
python auto_planner.py
```

⚠️ 이 스크립트는 종료 전까지 블로킹됩니다. 백그라운드에서 돌리려면 별도 터미널을 사용하세요.
