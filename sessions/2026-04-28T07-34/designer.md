## 1. 최종 디자인 시스템 확정 (Visual System Finalization)

신뢰감과 미래지향성을 동시에 표현하기 위해 **Deep Blue**를 기반으로 **Electric Neon Accent**를 사용하는 색상 팔레트와 타이포그래피 가이드라인을 확정합니다.

### A. 컬러 팔레트 (Color Palette)

| 역할 | 색상 코드 (Hex) | 설명 | 사용 목적 |
| :--- | :--- | :--- | :--- |
| **Primary (신뢰/기반)** | `#0A1931` (Deep Navy Blue) | 깊은 신뢰감과 전문성을 제공하는 메인 배경색. | 배경, 주요 텍스트 영역. |
| **Secondary (미래/강조)** | `#4169E1` (Royal Blue / Electric Blue) | 시스템의 논리와 흐름을 나타내는 주된 강조색. | 버튼, 핵심 그래프 라인, 링크 하이라이트. |
| **Accent (네온/에너지)** | `#00FFFF` (Cyan Neon) | 미래지향적이고 기술적인 에너지를 표현하는 네온 색상. | CTA 버튼 배경, 핵심 데이터 강조, 경고/성장 지표. |
| **Text (명료성)** | `#E6E6FA` (Light Lavender) | 어두운 배경에서 높은 가독성을 제공하는 밝은 텍스트 색상. | 본문 텍스트. |

### B. 타이포그래피 가이드라인 (Typography Guidelines)

시스템 설계자처럼 명료하고 구조적인 느낌을 주기 위해 기하학적이고 모던한 산세리프체를 사용합니다.

1.  **헤드라인 (H1, H2):** **'Space Mono'** 또는 유사한 기술적 느낌의 굵은 산세리프체 (예: Montserrat Bold/Black 또는 Space Mono).
    *   *목표:* 시스템 설계자로서의 명료함과 권위 부여.
2.  **본문 및 UI 텍스트:** **'Inter'** 또는 **'Roboto'** (깔끔하고 읽기 쉬운 범용성 높은 산세리프체).
    *   *목표:* 정보 전달의 명확성과 가독성 극대화.

### C. 스타일링 컨셉 요약

*   **배경:** Deep Navy Blue (`#0A1931`)를 사용하여 깊이감과 전문성을 확보합니다.
*   **강조:** Electric Cyan/Blue (`#00FFFF`, `#4169E1`) 네온 색상을 시스템 흐름(Flow), 데이터, 행동 유도 버튼에 집중적으로 사용하여 미래지향적인 에너지를 전달합니다.
*   **느낌:** 어둡고 깊은 우주(Deep Space) 속에서 빛나는 AI 시스템의 설계도를 보는 듯한 느낌을 목표로 합니다.

## 2. Hero Section 와이어프레임 (Wireframe Concept)

웹사이트의 첫 화면인 Hero Section은 **'시스템 설계자로서의 권위 확립'**과 **'핵심 가치 즉각 전달'**에 초점을 맞춥니다.

### 레이아웃 컨셉: 'System Blueprint Launchpad'

| 영역 | 구성 요소 | 디자인 및 목적 |
| :--- | :--- | :--- |
| **A. 헤더/로고 (Top Bar)** | 로고 (`아 빨랑~~ 회사`), 네비게이션 링크 (Blueprint, Flow, Results, About) | 최소화하여 시각적 공간 확보. Deep Navy 배경에 White 텍스트. |
| **B. 메인 비주얼 영역 (The Core Statement)** | **H1 타이틀:** 가장 강력한 핵심 메시지 제시 (예: "AI 워크플로우 자동화의 시스템 설계") | 중앙에 배치하며, H1 텍스트는 밝은 라벤더 또는 화이트로 강조하여 주목도를 높임. |
| **C. 서브 카피/가치 제안** | 강력한 후킹 문구 (Pain Point & Solution) | H1 아래에 짧고 명료하게, 시스템 설계의 이점을 설명. (예: "단순 이론이 아닌, 고수익을 위한 AI 자동화 프레임워크를 제공합니다.") |
| **D. 핵심 CTA 블록** | **Primary CTA 버튼:** '시스템 설계 로드맵 보기' 또는 '무료 워크플로우 템플릿 다운로드' | Cyan Neon 색상을 배경으로 사용하고, 버튼 테두리에 미세한 네온 효과를 주어 시각적 에너지를 극대화. |
| **E. 신뢰 구축 요소 (Social Proof)** | 하단에 작은 아이콘 또는 트로피 형태의 성공 지표 배치. | (예: '수익화 시스템 설계', 'AI 자동화 마스터') - 미래지향적인 라인 아트 스타일 사용. |
