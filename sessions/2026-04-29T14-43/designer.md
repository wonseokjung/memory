## 시스템 기초 설계 모듈 초기 목업 디자인 사양 (Deep Blue/Neon System)

본 산출물은 확정된 **Deep Blue/Neon** 디자인 시스템을 기반으로, '시스템 설계 방법론'의 핵심인 **시스템 기초 설계 모듈(System Foundation Design Module)**에 대한 시각적 프레임워크의 초기 목업 디자인 사양입니다. 이 사양은 영상 인트로 및 핵심 슬라이드에 즉시 적용 가능하도록 구체적인 색상 코드와 레이아웃을 제시합니다.

---

### 1. 전체 비주얼 시스템 정의 (Design System Summary)

| 요소 | 속성 | 상세 내용 |
| :--- | :--- | :--- |
| **메인 컬러** | Deep Blue | 신뢰감, 안정성 (배경 및 주요 텍스트) |
| **강조 컬러** | Neon Cyan | 시스템 강조, 핵심 지표, CTA (액션 유도) |
| **보조 컬러** | Dark Gray/Black | 텍스트 본문, 대비 극대화 |
| **폰트 스타일** | Sans-Serif (예: Pretendard 또는 Noto Sans 계열) | 현대적이고 논리적인 느낌 강조 |

### 2. 영상 인트로 화면 목업 디자인 사양 (Hook & Intro Screen Mockup)

**목표:** 시청자에게 '시스템 설계자'로서의 권위와 미래지향적인 신뢰감을 즉시 전달한다.

| 영역 | 디자인 요소 | 색상 코드 (Hex) | 타이포그래피/레이아웃 지침 |
| :--- | :--- | :--- | :--- |
| **배경** | 전체 배경 | `#0A192F` (Deep Blue Dark) | 깊은 밤하늘 같은 어두운 블루. 신뢰감 극대화. |
| **메인 타이틀** | "수익 엔진 아키텍처" | `#64FFDA` (Neon Cyan) | 크고 굵게 중앙 배치. Neon Cyan으로 시스템의 핵심을 강조. |
| **서브 타이틀** | "실행보다 설계가 돈을 부르는 시스템" | `#8892B0` (Light Blue/Gray) | 메인 타이틀 아래에 작게 배치. 차분하게 논리적 배경 제시. |
| **시각적 흐름** | 시스템 다이어그램 예고 | Neon Cyan 라인 사용 | 화면 중앙을 가로지르는 추상적인 네온 라인이 Input $\rightarrow$ Process $\rightarrow$ Output의 논리적 연결을 시각적으로 암시함. |
| **CTA 강조** | 시작 버튼/다음 단계 유도 | `#64FFDA` (Neon Cyan) | 하단에 Neon Cyan 박스를 배치하여 즉각적인 행동(클릭)을 유도. |

### 3. 핵심 슬라이드 목업 디자인 사양 (System Foundation Module Mockup)

**목표:** 시스템 설계 방법론의 논리적 구조를 시각화하여, 실행보다 '설계'가 우위에 있음을 증명한다.

#### A. 레이아웃 컨셉: 대칭형 흐름도 (Symmetrical Flow Diagram)

*   **레이아웃:** 화면을 세로 또는 가로로 명확히 분할하고, 좌측에서 우측으로 논리적 흐름(Input $\rightarrow$ Process $\rightarrow$ Output)이 끊김 없이 이어지도록 설계한다.
*   **배경:** Deep Blue (`#0A192F`) 배경을 유지하며, 흐름선과 박스 테두리에 Neon Cyan (`#64FFDA`)을 적용하여 대비를 극대화한다.

#### B. 모듈 구성 요소 디자인 상세

| 모듈 | 제목/내용 강조 | 시각적 처리 (스타일링) | 목적 |
| :--- | :--- | :--- | :--- |
| **Input (입력)** | 시스템의 시작점 | Deep Blue 박스, Neon Cyan 테두리 | 모든 수익 창출의 원천(자원/지식)을 명확히 정의. |
| **Process (처리)** | 엔진 아키텍처 설계 | 중앙에 배치된 복잡한 다이어그램 또는 프레임워크 템플릿 적용 | 실행이 아닌, 논리적 구조화 작업임을 강조. (가장 시각적으로 복잡하게 처리) |
| **Output (결과)** | 수익 증폭 결과 | Neon Cyan으로 채워진 최종 결과 박스 | 명확하고 폭발적인 성과를 강조하여 신뢰도 부여. |

#### C. 폰트 및 색상 적용 규칙

1.  **제목/핵심 키워드:** 항상 **Neon Cyan**을 사용하여 시선이 가장 먼저 집중되도록 한다.
2.  **설명 텍스트:** Deep Blue 배경 위에서 가독성을 확보하기 위해 **White** 또는 **Light Gray**를 사용한다.
3.  **시스템 연결선/강조선:** 모든 논리적 연결과 흐름은 **Neon Cyan**으로 처리하여 시스템의 작동 방식을 시각적으로 명확히 한다.
