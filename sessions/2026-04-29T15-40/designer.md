## 🎬 시스템 비주얼 스토리보드 및 디자인 사양 확정

Writer가 제시한 스크립트의 핵심 메시지(수익 엔진 아키텍처)와 확정된 브랜드 시스템(Deep Blue/Neon Cyan)을 기반으로, 유튜브 영상의 인트로 화면과 핵심 슬라이드의 시각적 흐름(Storyboard)에 대한 구체적인 디자인 사양을 최종 확정합니다. 모든 요소는 **신뢰감**과 **시스템적 우위**를 극대화하는 데 중점을 둡니다.

---

### 1. 📺 영상 인트로 화면 (Hook) 디자인 사양

**목표:** 심리적 딜레마를 던지며 시청자의 즉각적인 몰입(Stakes 극대화)을 유도하고, 시스템의 권위를 제시한다.

| 요소 | 시각 사양 | 상세 설명 및 코드 |
| :--- | :--- | :--- |
| **배경** | Deep Blue (깊고 신뢰감 있는 톤) | `#0A1931` (거의 검정에 가까운 짙은 네이비 블루) |
| **주요 타이틀** | 시스템 설계 포지셔닝 강조 | `Neon Cyan` |
| **텍스트 배치** | 중앙 정렬, 대비 극대화 | 명확하고 권위적인 느낌 부여. |
| **핵심 질문 (HOOK 1)** | "운전자가 되고 싶습니까, 아니면 자동차의 설계자가 되고 싶습니까?" | 폰트: Montserrat Bold / 크기: 매우 크게. 색상: `#00FFFF` (순수 네온 시안) |
| **시각 효과** | 동적 시스템 흐름 강조 | 화면 전체에 복잡하지만 정돈된 듯한 **Neon Cyan**의 선과 노드(Node)가 빠르게 움직이며 '시스템'을 암시하는 애니메이션 적용. |
| **CTA 준비** | 다음 단계로의 연결고리 | 하단에 작은 텍스트로 "엔진 설계 방식 공개"를 배치하고, 다음 슬라이드로 자연스럽게 이어지도록 디자인한다. |

### 2. ⚙️ 본문 핵심 슬라이드 (시스템 프레임워크) 디자인 사양

**목표:** 복잡한 실행 방법을 논리적이고 통제 가능한 '수익 엔진 아키텍처'로 단순화하여 제시하고, 시스템의 신뢰도를 시각적으로 입증한다.

#### A. 인트로: 시스템의 중요성 강조 슬라이드 (0:45 지점)

| 요소 | 시각 사양 | 상세 설명 및 코드 |
| :--- | :--- | :--- |
| **배경** | Deep Blue | `#0A1931` |
| **제목** | 수익 극대화 시스템 설계 4단계 | `Neon Cyan` (가장 큰 크기) |
| **핵심 메시지** | "성공은 노력의 문제가 아니라, 최적화된 시스템 설계의 문제입니다." | 중앙 배치. 서브 텍스트는 회색 계열로 신뢰감 유지. |
| **시각 요소** | 4단계 흐름 도식 (Flowchart)의 시작점 제시 | 네 단계(Input, Process, Output 등)가 순차적으로 연결되는 최소한의 구조만 먼저 시각화하여 기대감을 높인다. |

#### B. 핵심 단계별 상세 슬라이드 디자인 사양 (Step 1: Input 예시)

**목표:** 각 단계를 명확하게 구분하고, 입력(Input)이 어떻게 시스템의 신뢰도를 구축하는지를 논리적으로 보여준다.

| 요소 | 시각 사양 | 상세 설명 및 코드 |
| :--- | :--- | :--- |
| **섹션 제목** | 핵심 단계 1: 입력 (Input) - 자원과 조건 설정 | `Neon Cyan` + 시스템 번호 강조 (`Step 1`) |
| **배경** | Deep Blue | `#0A1931` |
| **레이아웃** | 좌측에 키워드, 우측에 시각적 예시 박스 배치 (2분할 구조) | 논리적 대비를 통해 정보 전달력을 높인다. |
| **키워드 강조** | **Input:** 자원(시간, 지식, 자본), 조건(심리 상태) 정의 | 키워드는 `Neon Cyan`으로 처리하고, 그 옆에 작은 아이콘(예: 시계, 돈, 뇌 모양)을 배치한다. |
| **시스템 신뢰도 강조** | "이것은 무작위가 아닌, 수학적 측정의 시작입니다." | 이 문구는 화면 하단에 작은 박스 형태로 `Neon Cyan`으로 삽입하여 시스템 설계 철학을 각인시킨다. |
| **CTA (다음 단계)** | 다음 단계로 이동 유도 버튼 | 버튼 형태를 사용하며 배경은 Deep Blue, 테두리 및 텍스트는 `Neon Cyan`으로 명확하게 강조한다. |

### 3. 🎨 최종 디자인 시스템 요약

*   **메인 컬러:** Deep Blue (`#0A1931`) - 신뢰, 깊이, 권위
*   **강조/액션 컬러:** Neon Cyan (`#00FFFF`) - 미래지향성, 시스템 흐름, 중요 지점 (CTA 및 핵심 데이터)
*   **타이포그래피:** Montserrat (Bold, Semi-Bold 중심) - 논리적이고 현대적인 느낌.
*   **레이아웃 원칙:** 정보의 계층 구조(Hierarchy)를 명확히 하며, 모든 시각 요소는 '시스템'이라는 프레임워크 내에서 완벽하게 정렬되어야 한다.
